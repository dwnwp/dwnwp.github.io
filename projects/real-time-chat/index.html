Real-time Chat Application

Ruby on Rails SQLite Hotwire

A real-time web chat application built using Ruby on Rails. The application allows users to securely register, create dynamic chat rooms, and exchange messages instantly without the need for manual page refreshes.

Key Features

  • Real-Time WebSockets: Utilizes ActionCable alongside Turbo Streams to instantly broadcast newly created rooms and messages to all connected clients, providing a seamless real-time experience.
  • SPA-like Interactions: Leverages Turbo Frames to handle seamless navigation between chat rooms and form submissions, delivering the speed and feel of a Single Page Application using purely server-side rendered HTML.
