CPSU Multi-Agent Chatbot

Golang Python Vue 3 Ollama Qdrant MinIO Redis RabbitMQ

Multi-Agent Chatbot for providing information and answering questions related to the Computer Department at Silpakorn University. Available on LINE and Facebook Messenger, supporting both text and image replies. Includes a website to manage the chatbot's knowledge base, allowing admin to upload, delete, or edit documents and images in real-time.

Powered by Local LLM: typhoon2.1-gemma3-12b, Embedding model: BGE-M3 via Ollama.

Multi-Agent System Architecture

  • RAG Queries: The Main Agent invokes tools to pull data from Qdrant as a knowledge base, then forwards it to the Summarize Agent to analyze and generate replies.
  • General & Image Replies: For general messages or queries requiring image response, the Main Agent directly utilizes the send_image_tool to reply with images.

Chatbot Features & Usage Examples

General FAQ

Answers generic questions (grades, registration, tuition). If a user asks about tuition, the bot analyzes and asks follow-up questions (e.g. study plan, degree level) to ensure the answer matches perfectly.

Image Replies

Responds with maps, academic calendars, or study plan documents dynamically based on request context.

LLM Guardrails

Filters and blocks prompts outside scope of academic services to keep chatbot professional and secure.

Session Lock & Multi-Turn Messaging

Uses Session Lock + Message Queue. Consecutive messages are queued in Redis first, then read as a batch once AI finishes processing, providing complete context comprehension and saving resources.

Rate Limiting

Limits users to 5 messages per minute to prevent resource exhaustion and server abuse.

Stateful Conversations

Saves short-term conversation state so users can easily reference previous statements without re-entering parameters.

Knowledge Base Management Dashboard

JWT Authentication

Secure login using JSON Web Tokens.

Document Management

Allows search, edit and delete operations.

Real-time Upload

Adds PDFs, txt directly to chatbot knowledge base.

Image Library Management

Manage inline reply images such as study schedules and campuses map.
